## 🎯 Tópicos Abordados

- ✅ Arrays e Listas
- ✅ Ordenação (Sorting)
- ✅ Busca (Search)
- ✅ Strings
- ✅ Greedy Algorithms
- ✅ Dynamic Programming (Básico)
- ✅ Stacks e Queues
- ✅ Hash Maps / Dicionários
- ✅ BFS/DFS Simples

## 💡 Como Usar

1. Leia o **enunciado.md** de cada exercício
2. Veja os **exemplos** de entrada e saída
3. Implemente a solução na linguagem de sua escolha
4. Confira com as **soluções** na pasta SOLUCOES

## 🚀 Começar

**Nível 1** é perfeito para quem está começando. Depois suba para o **Nível 2** e **Nível 3**.
