# 01 - Soma Simples

## Descrição

Faça um programa que leia dois números inteiros e imprima a soma deles.

## Entrada

Uma linha contendo dois inteiros **A** e **B**.

## Saída

Um único inteiro representando a soma de **A** e **B**.

## Exemplos

### Exemplo 1
**Entrada:**
```
2 3
```
**Saída:**
```
5
```

### Exemplo 2
**Entrada:**
```
10 20
```
**Saída:**
```
30
```

### Exemplo 3
**Entrada:**
```
-5 10
```
**Saída:**
```
5
```

## Constraints

- `-1000 <= A, B <= 1000`

## Dicas

💡 Este é um problema básico para aquecimento. Pratique a leitura de entrada e saída.
