# 02 - Maior Número

## Descrição

Leia três números inteiros e mostre qual deles é o maior.

## Entrada

Uma linha contendo três inteiros **A**, **B** e **C**.

## Saída

Um único inteiro representando o maior dos três números.

## Exemplos

### Exemplo 1
**Entrada:**
```
7 13 9
```
**Saída:**
```
13
```

### Exemplo 2
**Entrada:**
```
100 50 75
```
**Saída:**
```
100
```

### Exemplo 3
**Entrada:**
```
-10 -5 -20
```
**Saída:**
```
-5
```

## Constraints

- `-1000 <= A, B, C <= 1000`

## Dicas

💡 Use `if` para comparação ou a função `max()` de sua linguagem.
